# Local Memory

This folder is for agent-written operational memory. Runtime memory files are ignored by git.

Read `core/memory/GUIDE.md` before reading or writing here.

Suggested files:

```text
memory/index.md
memory/environment.md
memory/failures.md
memory/devices/<device-alias>.md
memory/apps/<app-id>.md
```

Never store credentials, tokens, OTPs, payment data, private messages, contacts, or screenshots.
